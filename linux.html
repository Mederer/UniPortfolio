Linux

I am big fan of the Linux operating system, and one of my hobbies include spending a lot of time tinkering and playing around with various Linux distributions (I'm what some would call a 'distro hopper'). I have a lot of familiarity with the Linux terminal, the Linux filesystem, as well as Bash scripting.

The various distributions I've used include:

  • Ubuntu
  • CentOS
  • Kali Linux
  • Red Hat Enterprise Linux
  • Fedora
  • Arch Linux

I also have some experience using IBM AIX, a Unix based operating system that shares many similarities with Linux. I have "AIX System Administrator Fundamentals" certificate, issued by IBM.
